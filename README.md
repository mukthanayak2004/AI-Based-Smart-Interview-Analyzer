# 🤖 AI-Based Smart Interview Analyzer

## 📌 Overview
AI-Based Smart Interview Analyzer is an intelligent web application that helps evaluate candidates during interviews using Artificial Intelligence. The system analyzes candidate responses and generates performance feedback.

---

## 🎯 Problem Statement
Traditional interviews often rely only on manual evaluation, which can be subjective and time-consuming. This project aims to provide an AI-assisted interview evaluation system.

---

## ✨ Features
- Candidate Login
- AI-Based Question Generation
- Speech-to-Text Response Analysis
- Performance Evaluation
- Strengths & Weaknesses Report
- Interview Score Generation
